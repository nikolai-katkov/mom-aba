# 01-02: Экран списка разделов

Статус: Черновик

## Описание

Точка входа в приложение, показывающая доступные разделы навыков в виде карточек. МАНД активен с отслеживанием прогресса. ТАКТ отображается как неактивная заглушка "скоро будет". Нажатие на МАНД переходит к экрану введения в МАНД.

## Критерии приёмки для пользователя

- [ ] Карточка МАНД отображает: заголовок "МАНД", подзаголовок "Просьбы", статус (Не начато / В процессе / Завершено), прогресс (X/5)
- [ ] Карточка ТАКТ отображает: заголовок "ТАКТ", подзаголовок "Скоро будет", визуально неактивна/затемнена
- [ ] Нажатие на карточку МАНД переходит к экрану введения в МАНД
- [ ] Нажатие на карточку ТАКТ не выполняет действий (неактивна)
- [ ] Обновления прогресса отражают фактическое состояние оценки (сохраняется между сеансами)

## Системные критерии приёмки

- [ ] Маршрут: `/` (точка входа в приложение)
- [ ] Прогресс раздела вычисляется из состояний критериев (количество завершённых из общего числа)
- [ ] Статус раздела определяется: все NotStarted = "Не начато", любой InProgress или смешанный = "В процессе", все Completed = "Завершено"
- [ ] Состояние сохраняется между перезагрузками страницы (localStorage или аналог)

## Критерии приёмки для разработки

- [ ] CSS Modules только с дизайн-токенами (без захардкоженных значений)
- [ ] Адаптивность для десктопа (1920x1080), планшета (768x1024), мобильного (375x667)
- [ ] Юнит-тест для логики вычисления прогресса/статуса
- [ ] E2E-тест: загрузка приложения, проверка видимости карточки МАНД, нажатие переходит к введению
