# 01-08: Выпадающие меню навигации в хлебных крошках

Статус: Готово

## Описание

Выпадающие меню по наведению на сегменты хлебных крошек на десктопе, у которых есть соседние страницы (разделы и критерии). Позволяет переключаться между разделами или критериями без возврата назад. На узких экранах вместо полной цепочки отображается одна ссылка «назад».

## Критерии приёмки для пользователя

- [x] Сегмент раздела (например, «Манд») показывает выпадающее меню со всеми доступными разделами по наведению
- [x] Сегмент критерия (например, «III - Обобщение 6 просьб») показывает выпадающее меню со всеми критериями раздела по наведению
- [x] Текущая страница выделена в меню (жирный шрифт + основной цвет)
- [x] Выбор соседней страницы переводит на тот же тип страницы (оценка -> оценка, тренировка -> тренировка)
- [x] Сегменты без соседних страниц (Главная, Уровни, Тренировка) не имеют выпадающего меню
- [x] На узких экранах (<=640px) полная цепочка сворачивается в одну ссылку «назад»

## Системные критерии приёмки

- [x] Меню открывается с задержкой 150мс, закрывается с задержкой 300мс
- [x] Клавиатурная навигация: Enter/Space переключает, ArrowDown/ArrowUp перемещает по пунктам, Escape закрывает
- [x] ARIA: `aria-haspopup="menu"`, `aria-expanded`, `role="menu"`, `role="menuitem"`, `aria-current="location"`
- [x] Меню рендерится поверх контента страницы (z-index на header)

## Критерии приёмки для разработки

- [x] Все стили используют дизайн-токены — автоматическая адаптация ко всем 3 темам (Тёплая/Мягкая/Элегантная) x 2 режима
- [x] Viewport-брейкпоинт (640px) управляет цепочкой vs ссылкой «назад» — не UA и не детекция тач
- [x] Размеры тач-целей через `@media (hover: none) and (pointer: coarse)` — не viewport
- [x] Без именования «mobile» в CSS — используется терминология viewport/pointer
- [x] `BreadcrumbDropdown` внутренний для директории Breadcrumbs (не экспортируется из barrel)
- [x] `BreadcrumbItem` расширен опциональным `siblings` (обратная совместимость)
- [x] ESLint и TypeScript без ошибок
- [x] Все существующие тесты проходят
