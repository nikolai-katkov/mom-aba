# 01-06: Экран тренировки

Статус: Черновик

## Описание

Экран с руководством, показываемый после оценки "Нет". Обучает родителя тому, как развивать конкретный навык, используя прогрессивное раскрытие: заглушка видео, краткое пошаговое руководство видно по умолчанию, полное руководство свёрнуто. "Повторить" возвращает к экрану оценки для повторной проверки. "Назад" возвращает к списку критериев.

## Критерии приёмки для пользователя

- [ ] Место для видео отображает статичное изображение-заглушку с надписью "Видео скоро будет"
- [ ] Краткое руководство видно по умолчанию: пронумерованные шаги (например, 1. Покажите 2 предмета, 2. Подождите, 3. Подскажите при необходимости, 4. Подкрепите)
- [ ] "Подробнее" разворачивает полное руководство (детальное объяснение, распространённые ошибки, дополнительные примеры)
- [ ] Сворачивание "Подробнее" скрывает полное руководство
- [ ] Кнопка "Повторить" переходит назад к экрану оценки для этого критерия
- [ ] Кнопка "Назад" возвращает к списку критериев
- [ ] Тренировочный контент специфичен для критерия (не общий)

## Системные критерии приёмки

- [ ] Маршрут: `/sections/:sectionId/criteria/:criterionId/train`
- [ ] Тренировочный контент загружается из отдельной модели данных тренировки (не из данных критерия)
- [ ] "Повторить" переходит к экрану оценки с сохранением контекста критерия
- [ ] "Назад" переходит к списку критериев
- [ ] Критерий остаётся в статусе InProgress во время цикла тренировки
- [ ] Нет тупиков: обе кнопки всегда видны и функциональны

## Критерии приёмки для разработки

- [ ] Прогрессивное раскрытие использует CSS-переходы (согласовано с паттерном экрана введения)
- [ ] CSS Modules только с дизайн-токенами
- [ ] Адаптивность для всех профилей устройств
- [ ] Повторное использование компонента/паттерна прогрессивного раскрытия из 01-03, если применимо (правило двух -- следить за третьим использованием)
- [ ] Юнит-тест: поведение переключателя "Подробнее"
- [ ] Юнит-тест: корректные данные тренировки загружаются на основе параметров маршрута
- [ ] E2E-тест: переход из оценки "Нет", проверка тренировочного контента, нажатие "Повторить" возвращает к оценке
- [ ] E2E-тест: "Назад" возвращает к списку критериев
