# Эпик 01: Основа оценки

## Описание

Реализация основного потока оценки для раздела МАНД: список разделов, введение, прогрессия критериев, бинарная оценка, руководство по тренировке и цикл оценка-тренировка-переоценка. ТАКТ отображается как заглушка.

**Поток:** Изучение > Оценка > (при необходимости) Тренировка > Переоценка
