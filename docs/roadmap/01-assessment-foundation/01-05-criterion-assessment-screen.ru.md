# 01-05: Экран оценки критерия

Статус: Черновик

## Описание

Экран бинарной оценки для одного критерия. Представляет чёткий вопрос Да/Нет с иллюстрацией и контекстной информацией. "Да" отмечает критерий как завершённый и возвращает к списку критериев. "Нет" переходит к экрану тренировки.

## Критерии приёмки для пользователя

- [ ] Чёткий бинарный вопрос отображается на видном месте (например, "Может ли ребёнок попросить один из двух предметов?")
- [ ] Область иллюстрации показывает статичное изображение-заглушку, релевантное критерию (2 различных объекта для критериев МАНД)
- [ ] Раздел контекста показывает: краткое объяснение того, на что обращать внимание, условия из критерия
- [ ] Две кнопки: "Да" и "Нет" -- без двусмысленности, без других вариантов
- [ ] "Да" показывает краткую положительную обратную связь, отмечает выполнение, возвращает к списку критериев с обновлённым статусом
- [ ] "Нет" переходит к экрану тренировки для этого критерия
- [ ] Условия критерия (например, "подсказки не допускаются") видны, чтобы родитель оценивал правильно

## Системные критерии приёмки

- [ ] Маршрут: `/sections/:sectionId/criteria/:criterionId/assess`
- [ ] "Да" сохраняет балл (1) и обновляет статус критерия на Completed
- [ ] "Нет" сохраняет балл (0) и обновляет статус критерия на InProgress
- [ ] Балл хранится как числовое значение (0 или 1), не булево (см. [Вопросы PM: Модель оценивания](../../knowledge/pm-questions-scoring-model.ru.md))
- [ ] Состояние сохраняется между перезагрузками страницы
- [ ] Навигация: Да -> список критериев, Нет -> экран тренировки

## Критерии приёмки для разработки

- [ ] CSS Modules только с дизайн-токенами
- [ ] Адаптивность для всех профилей устройств
- [ ] Изображения-заглушки используют единое соотношение сторон и стиль по умолчанию
- [ ] Юнит-тест: обработчики Да/Нет корректно обновляют состояние
- [ ] Юнит-тест: корректные данные критерия загружаются на основе параметров маршрута
- [ ] E2E-тест: полный поток "Да" (нажатие Да -> возврат к списку с обновлённым состоянием)
- [ ] E2E-тест: полный поток "Нет" (нажатие Нет -> переход к экрану тренировки)
